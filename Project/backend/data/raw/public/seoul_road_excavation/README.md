# 서울시 도로굴착 공사 정보 적재 폴더

이 폴더에 서울시 도로굴착 공사 정보 파일을 넣으면 백엔드가 주기적으로 읽어
`construction_events`에 적재하고 위험도 점수의 `공사 영향` 항목에 반영합니다.

지원 확장자:

- `.csv`
- `.tsv`
- `.txt`
- `.xlsx`
- `.xls`

권장 컬럼:

- 공사명 또는 공사종류
- 도로명주소 또는 공사위치
- 시작일 또는 착공일
- 종료일 또는 준공일
- 굴착연장, 굴착깊이, 굴착폭, 굴착면적 중 가능한 값
- 위도, 경도

파일을 교체하거나 삭제하면 다음 스캔 때 기존 적재분도 갱신됩니다.
