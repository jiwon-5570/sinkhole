언어 설정

모드 전환

시나리오 모드는 내부 데이터, 실시간 모드는 입력 위치를 기반으로 분석합니다.

  • 시나리오 모드는 사전 구축된 지역 및 도로 데이터를 사용합니다.
  • 실시간 모드는 위치명을 입력해 상용 분석 API를 실행합니다.

지도 안내

위험도 색상은 낮음, 중간, 높음, 매우 높음 단계로 표시됩니다.

지도 위 점을 클릭하면 해당 지역 분석을 바로 실행합니다.

분석 지표 설명

  • 과거 지반침하: 발생 빈도와 규모를 반영합니다.
  • GPR: 지중 공동 탐사 결과를 반영합니다.
  • 시설물: 지하시설물 노후도와 상태를 반영합니다.
  • 강우량: 최근 강우로 인한 지반 약화 가능성을 반영합니다.
  • 지하수: 지하수위 변동을 반영합니다.

지표를 종합해 최종 위험 점수(0~100)를 산출합니다.
