# 02_공공데이터_학습_가이드.md

# 국토안전관리 프로젝트 공공데이터 활용 및 학습 가이드

## 1. 문서 목적

이 문서는 국토안전관리 프로젝트에서 사용할 공공데이터의 종류, 수집 방법, 전처리 방식, 학습 데이터 구성, 데이터 품질 관리, 모델 학습용 데이터셋 구성 방법을 정의한다.

본 가이드는 다음을 목표로 한다.

1. 국토안전관리원 중심 데이터 구조를 명확히 정의한다.
2. 외부 공공데이터의 역할과 결합 방법을 체계화한다.
3. 학습 가능한 형태로 데이터를 변환하는 기준을 제시한다.
4. 모델 입력 데이터(feature dataset) 생성 기준을 정의한다.
5. 데이터 기반 서비스의 신뢰성과 설명 가능성을 확보한다.

---

## 2. 데이터 활용 기본 원칙

### 2.1 국토안전관리원 데이터 중심
이 프로젝트는 반드시 다음 원칙을 따른다.

- 핵심 데이터는 국토안전관리원 데이터
- 분석 기준은 지반침하 및 지하안전 데이터
- 외부 데이터는 보조적으로 활용

### 2.2 데이터 역할 분리

데이터는 아래 3가지 역할로 구분한다.

1. 기본 취약도 데이터
2. 위험 상승 요인 데이터
3. 설명 및 보조 데이터

이 프로젝트의 데이터 구조는 다음과 같다.

중심: 국토안전관리원 데이터

→ 위험도를 설명하기 위한 구조로 재구성되어야 한다.
